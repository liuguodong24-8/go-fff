##交易nft

### 1) 请求地址

>api/trans-nft

### 2) 调用方式：HTTP POST

### 3) 接口描述：

* 上传nft

### 4) 请求参数:

#### GET参数:
| 字段名称          | 字段说明    | 类型     | 必填 | 备注   |
|---------------|---------|--------|----  |------|
| nft_id        | NFTid   | int    | Y |  |
| address       | 转入的对方地址 | string | Y |  |


### 5) 请求返回结果:

```
{
    "code": 0,
    "msg": "交易NFT哈希成功"
}
```

### 6) 请求返回结果参数说明:
| 字段名称   | 字段说明                | 类型     | 备注 |
|--------|---------------------|--------|----  |
| code   | 0成功 1参数错误 2数据为空 3失败 | int    | - | 
| msg |                     | string | - |
